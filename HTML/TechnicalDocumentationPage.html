This documentation provides an overview of core web technologies and programming languages used in modern development.

Each section explains the basics, key features, and example code snippets.

  • Java for backend development
  • HTML for structuring web pages
  • CSS for styling
  • JavaScript for interactivity
// Example: Hello World in multiple languages
Java

Java is a versatile, object-oriented programming language widely used for building enterprise applications, Android apps, and backend systems.

It emphasizes portability, performance, and security.

  • Strongly typed language
  • Runs on the JVM
  • Supports multithreading
public class HelloWorld { public static void main(String[] args) { System.out.println("Hello, World!"); } }
HTML

HTML (HyperText Markup Language) is the standard language for creating web pages.

It defines the structure and layout of content using elements and tags.

  • Defines headings, paragraphs, lists
  • Supports links and images
  • Forms for user input

Hello World

This is a paragraph.
CSS

CSS (Cascading Style Sheets) is used to style and format HTML elements.

It controls layout, colors, fonts, and responsiveness.

  • Selectors target HTML elements
  • Supports animations and transitions
  • Media queries for responsive design
h1 { color: blue; text-align: center; } p { font-size: 16px; }
JavaScript

JavaScript is a dynamic scripting language that adds interactivity to web pages.

It runs in the browser and can manipulate the DOM, handle events, and communicate with servers.

  • Client-side scripting
  • Supports ES6+ features
  • Works with APIs
document.addEventListener("DOMContentLoaded", () => { alert("Hello, World!"); });
